Вы закончили работу над задачей для клиента у вас есть рабочий код, который который был создан без использования git. Клиент создал пустой репозирорий git@example.com:example/test.git и расшалил для вас доступ.

Задание
Опишите вашы дальнейшие действия что бы ваш код появился на ветке master в репозитории git@example.com:example/test.git


1. Открываем Git Bash и меняем текущую папку на папку с проектом, который нужно добавить 

2. Инициализируем папку с проектом как гит-репозиторий:

$ git init -b main

3. Командой git add добавим файлы в новый репозиторий

4. Сделаем первый коммит для новых файлов

$ git commit -m "First commit"


5. Добавляем ссылку на удаленный пустой репозиторий, созданный клиентом как место назначения куда будет выгруженны наши файлы

$ git remote add origin https://github.com/git@example.com:example/test.git
$ git remote -v

6. Пушим в репозиторий

$ git push origin master
